# dotfiles-minimal

## Requirements

- Vim 9.1+

## Installation

```powershell
git clone https://github.com/kumarstack55/dotfiles-minimal.git

Set-Location .\dotfiles-minimal

Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass

.\Invoke-Installer.ps1 -WhatIf

.\Invoke-Installer.ps1
```
